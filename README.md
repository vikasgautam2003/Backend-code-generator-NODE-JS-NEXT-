## üìñ About The Project

BackNest.ai is an AI backend-generation platform that transforms plain-English prompts into fully structured backend codebases.  
You describe your API logic ‚Äî BackNest generates:

- Modular backend architecture  
- Express or Next.js API routes  
- Models, controllers, middleware, services  
- DB logic for MongoDB, Prisma, PostgreSQL, Sequelize  
- JWT auth, validation, utilities  
- Multi-file folder hierarchy  

Every file appears as a separate tab in the UI, making it easy to explore and export your generated backend.

---

## üåü Key Features

- **AI-Generated Modular Backends**  
  Creates controllers, routes, services, models, middleware, validations, configs ‚Äî everything.

- **Multi-File Output Engine**  
  Produces a complete folder tree like:  
  `routes/`, `controllers/`, `models/`, `middleware/`, `services/`, `utils/`.

- **Runtime Intelligent**  
  Generates correct code for Express.js or Next.js App Router depending on your selection.

- **Database Intelligent**  
  Generates DB-specific implementations for:  
  - Mongoose  
  - Prisma  
  - PostgreSQL (pg)  
  - Sequelize  

- **One-Click Copy + Download**  
  Export individual files or full backend bundles instantly.

- **Beautiful Developer UI**  
  Tab-based file viewer, smooth scrolling, and a polished dev-tool aesthetic.
